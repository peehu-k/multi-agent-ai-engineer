# Multi-Agent Autonomous AI Software Engineer

A self-improving multi-agent AI system that autonomously generates, debugs, optimizes, benchmarks, and evolves production-ready Python tools using local LLMs.

## Key Features
• Multi-agent architecture (Coder, Debugger, Reviewer, Improver)  
• Autonomous tool generation & reuse  
• Self-evolving codebase  
• Benchmarking engine with performance tracking  
• Real-time Streamlit intelligence dashboard  
• Tool ecosystem + memory learning  
• Architect mode for full system design  

## Performance
• Success Rate: ~100% across benchmark tasks  
• Avg Code Quality: ~8.5/10  
• Autonomous tool building + optimization  
• Self-improving intelligence scoring system  

## Run locally
python agent.py
streamlit run dashboard.py
